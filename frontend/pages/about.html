About This Project

This Advanced CPU Scheduling Simulator is an Operating System project that demonstrates how CPU scheduling algorithms allocate CPU time to processes.

Algorithms Implemented

  • FCFS
  • SJF
  • Round Robin
  • Priority Scheduling

Advanced Concepts

  • Context switching overhead
  • Starvation detection
  • Aging mechanism
  • Ready queue visualization
  • Gantt chart execution timeline
